# Malina Pasrser

Парсер интернет-магазина одежды.
Собирает данные о штанах: название, цена, наличие.

## Стек
 * Python 3.10
 * requests
 * BeautifulSoup

## Возможности
- Парсинг списка товаров
- Поддержка пагинации
- Сохранение данных в CSV / JSON

## Установка
```bash
pip install -r requirements.txt
```
